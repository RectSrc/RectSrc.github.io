rpm docs

This documentation will guide you through the usage and installation of rpm. It will also teach you about making your package.


Installation

How to install rpm:
  • Go to the downloads page and download the latest verison
  • Extract the files in the zip directly into your ReCT folder, rpm.exe and Rect-ide.exe should be in the same directory
  • You are now done!

First time usage

rpm might be tricky in the beginning, but it will be fine. It is fairly simple to get used to.

Testing rpm

  • Navigate to the folder containing rpm.exe
  • Open cmd and run the command 
    cd rpm.exe-path-here
  • Now do 
    rpm
    You should see an output containing some info about rpm
  • Great! Now do 
    rpm get test
    It should have sucessfully installed a package named test
  • Now find a folder called packages, that will contain test.dll and test-DEPS, if not, please report an issue at the rpm github

rpm commands

rpm get [name]
Downloads the package [name] 

rpm remove [name]
Removes the package [name] 

rpm
Shows commands

Making and submitting a basic package

So, you feel like you want to create a package for rpm, then you must first complete these simple steps
You will need:
  • Some C# knowledge
  • Some knowledge in ReCT package making, more on rect.ml
  1. Create your standard ReCT package
  2. In the same folder as the package, also add a folder called [packagename]-DEPS, in wich you replace [packagename] with your packages .dll name
  3. Also add a file called deps.dps
  4. Now, if you need to add any dependecies, drop them into [packagename]-DEPS, and then add the filename to deps.dps, seperate with new lines
  5. To submit, please add [packagename].dll, deps.dps and [packagename]-DEPS into a zip file
  6. Then, finally fill in this google form and make sure to upload your .zip
