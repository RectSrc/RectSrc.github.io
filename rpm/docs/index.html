rpm docs

This documentation will guide you through the usage and installation of rpm. It will also teach you about making your package.


Installation

How to install rpm:
  • Go to the downloads page and download the latest verison
  • Extract the files in the zip directly into your ReCT folder, rpm.exe and Rect-ide.exe should be in the same directory
  • You are now done!

First time usage

rpm might be tricky in the beginning, but it will be fine. It is fairly simple to get used to.
