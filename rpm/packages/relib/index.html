relib

rpm install relib
Relib is a port of Raylib for ReCT, it allows for full 2D and 3D rendering.

GETTING STARTED

Windows: if youre on Windows you will need to inlude these files 
package dll Relib; use Relib;

#copy("Raylib-cs.dll");
#copy("raylib.dll");
Linux: on linux you'll need these files: 
package dll Relib; use Relib;

#copy("Raylib-cs.dll");
#copy("libraylib.so");
tip! -> you can include both raylib.dll and libraylib.so and it will work on both plattforms! Simple start script 
package dll Relib; use Relib;

#copy("libraylib.so");
#copy("Raylib-cs.dll");

Relib::InitWindow(500, 500, "Relib window");
Relib::SetTargetFPS(60);

while(!Relib::WindowShouldClose())
{
    Relib::BeginDrawing();
    Relib::ClearBackground(make Color(0, 0, 0, 255));

    Relib::EndDrawing();
}

Relib::CloseWindow();
this will create a 500x500 pixel black window
